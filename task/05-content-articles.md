  ```toml
  date = 2026-04-16
  draft = false

  +++
  ```

## Критерий готовности

- Для каждого раздела есть минимум одна дочерняя статья (чтобы раздел не был пустым).
- Все статьи из черновика созданы и опубликованы (`draft = false`).
- Хлебные крошки на статье отображают путь: Главная → Раздел → Статья.

## Заметка

Фазу можно выполнять итеративно: не обязательно закрывать все статьи разом. Но пустой раздел (без дочерних страниц) выглядит плохо — минимум одну статью в каждый.
