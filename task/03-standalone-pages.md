## Шаги

- [ ] **3.1.** Для каждой страницы создать `content/<slug>.md` (в корне `content/`, чтобы URL был `/<slug>.html` благодаря `uglyURLs`).
- [ ] **3.2.** Front matter для всех:
  ```toml
  +++
  title = '<Название>'
  date = 2026-04-16
  draft = false
  description = '<meta description, 120–160 символов>'
  showOnMain = false
  +++
  ```
- [ ] **3.3.** Заготовка тела: H2 «Когда нужна услуга», «Что входит», «Стоимость/бесплатно», CTA с телефоном.
- [ ] **3.4.** Пункты меню «Консультации», «Представительство в суде», «Подготовка документов» (из фазы 2) должны указывать на соответствующие страницы.
- [ ] **3.5.** Проверить все ссылки на главной и в футере.

## Критерий готовности

- Все 7 страниц доступны по своим URL, отдают 200.
- Пункты главного меню и футера ссылаются на рабочие страницы.
- Нет дубликатов: либо `yuridicheskaya-konsultaciya.html` остался, либо заменён на `konsultaciya.html` (с redirect, если нужно).
