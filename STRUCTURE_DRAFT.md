# Черновик структуры сайта 89095688888.ru

## Принятые решения (2026-04-16)

- Деление на физ/юр лиц НЕ нужно — практика ориентирована на граждан
- Разделы выносятся на главную (механизм showOnMain уже работает)
- Структура строится по направлениям практики, а не по типу клиента
- Жилищные и земельные споры — две отдельные категории
- Двухуровневая структура: разделы (по типам правоотношений) + статьи внутри (по конкретным проблемам клиента)

### Дополнительные решения (фаза 0 плана, 2026-04-16)

- **URL-схема:** во всех слагах используется дефис (`-`), не подчёркивание. Существующие `nasledstvennye_spory` → `nasledstvo`, `zashhita_prav_potrebitelja` → `zashhita-prav-potrebitelej`, `yuridicheskaya-konsultaciya` → `konsultaciya`.
- **Долги/кредиты:** остаются двумя standalone-страницами (разная аудитория — кредитор vs должник). В раздел объединяем, если появится 3-я и более тем.
- **Медорганизации:** внутри раздела «Защита прав потребителей», а не отдельно.
- **«О юристе»:** создаём.
- **«Контакты»:** создаём, без карты (адрес + телефон текстом), чтобы не тянуть сторонние cookies.
- **Политика конфиденциальности и Условия использования:** создаём в рамках этой итерации.
- **Сводные точки входа:** комбинация «главная (`showOnMain`) + хаб `/napravleniya.html` + блок в футере». В главное меню добавляется 4-й пункт «Направления практики».
