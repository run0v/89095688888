## Критерий готовности

- На мобильном один тап по номеру — идёт вызов.
- Кнопка «Записаться» видна на всех экранах без скролла.
- При скролле шапка остаётся сверху и не прыгает.

## Риск

Sticky + мобильный адресной бар iOS Safari = потенциальные баги с `100vh`. Использовать `100dvh` / `100svh` там, где нужно.
