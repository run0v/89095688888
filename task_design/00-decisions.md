### 1. Акцентный цвет 🟡

Варианты:
- (a) оставить коралловый, но использовать его только на CTA;
- (b) заменить на тёплую охру/золото (ближе к юридической классике);
- (c) глубокий navy + белый + один акцент (минимализм).

**Решение (заглушка):** **(b) охра/золото на CTA + глубокий navy как база.**

Палитра:
- `--brand: #1F2A44` (тёмный navy — шапка, футер, заголовки)
- `--brand-dark: #141B2E`
- `--surface-0: #FFFFFF` (фон страницы)
- `--surface-1: #F5F3EE` (светлый тёплый — карточки, чередование секций)
- `--accent: #B8860B` (тёмная охра/золото — CTA-кнопки, ссылки, акценты)
- `--accent-dark: #8B6508` (hover)
- `--text-primary: #1F2A44`
- `--text-secondary: #4A5568`
- `--text-muted: #718096`
- `--border: #E2E0DA`

Почему: коралловый смотрится «стартапно», охра даёт классический юридический тон, сине-серый сохраняется как база.

### 5. Бэкенд формы заявки 🟡

**Решение (заглушка):** **(b) Telegram-бот через Cloudflare Worker.**

Почему:
- бесплатно (CF Workers free tier: 100k запросов/день);
- мгновенные уведомления юристу на телефон;
- Telegram-бот быстрее поднять, чем SMTP;
- юрист и так в Telegram — не нужно учить новый инструмент.

Схема:
1. Форма `POST /api/lead` → Cloudflare Worker.
2. Worker проверяет honeypot, rate-limit (по IP через KV), валидирует поля.
3. Успех → `sendMessage` в Telegram-чат юриста (бот + chat_id в env).
4. Неуспех → 400 с текстом ошибки.

Fallback: если CF Worker недоступен — форма с `action="mailto:..."` как last resort (graceful degradation).

Альтернатива (если не хотим свой код): Formspark за $5/мес. Оставляем на случай если юрист не захочет возиться с Telegram-ботом.

### 10. Адрес приёма 🟡

**Решение (заглушка):** **приём по договорённости (без постоянного офиса) + онлайн.**

Почему: у частнопрактикующих юристов аренда офиса часто нецелесообразна — встречи в кафе / МФЦ / у клиента / онлайн.

В `Schema.org` → не использовать `address`, вместо этого:
- `areaServed: { "@type": "City", "name": "Москва" }` (или реальный город).
- `serviceType: "Юридические услуги"`.
- `availableChannel: { "@type": "ServiceChannel", "serviceUrl": "https://89095688888.ru/zayavka.html" }`.

На странице `/kontakty.html`:
- «Встречи — по предварительной договорённости. Онлайн-консультации по WhatsApp/Telegram.»
- Без карты (карта создаёт ложное ожидание офиса).

⚠️ Если у юриста **есть** офис — заглушка неверна, нужен полный адрес.

### 12. uglyURLs 🟡

**Решение (заглушка):** **оставляем `.html` как есть.**

Почему:
- Сайт уже несколько лет с `.html`-путями, смена = пересчёт sitemap, все внешние ссылки (если есть) ломаются.
- 301-редиректы на GitHub Pages нетривиальны (нужен Cloudflare-прокси или Netlify).
- Эстетически чистые URL красивее, но это копеечный выигрыш по сравнению с «критичными» фазами (hero, форма, trust).

Если в будущем мигрируем на Cloudflare Pages — вернуться к вопросу и сделать за один коммит + `_redirects`.
