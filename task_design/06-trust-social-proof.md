
### Шаги

- [ ] **6.10.** Partial `floating-contacts.html`. Подключается в [baseof.html](../layouts/baseof.html) — выводится на всех страницах, кроме `/privacy-policy` и `/terms-of-service` (проверка `.Type` / path).
- [ ] **6.11.** Кнопки — SVG-иконки, размер 48–56px (тап-зона), фоновый цвет бренд-мессенджера (WhatsApp зелёный, Telegram синий) либо нейтральный.
- [ ] **6.12.** Ссылки:
  - `https://wa.me/79095688888?text=...`
  - `https://t.me/<username>` или `tg://resolve?domain=...`
  - `tel:+79095688888`
- [ ] **6.13.** На мобильном — показываем всегда; на десктопе — опционально, учитывая что в шапке уже есть CTA. Решить в процессе A/B.
- [ ] **6.14.** Анимация появления (через 3 сек или при скролле > 300px) — опционально, без JS-необходимости обойтись CSS-анимацией на `animation-delay`.

- [ ] **6.20.** Логотипы систем (с разрешения): «Палата адвокатов …», «Член …» — если применимо.

## Критерий готовности

- На главной: 3 отзыва + 2 кейса + блок «Обо мне».
- Плавающие мессенджеры работают, мобильная версия поднимает родное приложение.
- Schema.org Review / AggregateRating проходит валидацию.

## Риск

Поддельные отзывы = гарантированный минус в доверии и риск по 38-ФЗ. Использовать только реальные, с указанием источника и возможностью верифицировать.
